
Appium安装配置 
=============================

下载地址
------------------------------

https://bitbucket.org/appium/appium.app/downloads/

Mac安装
----------------------------

.. code-block:: python

    > brew install node      # get node.js
    > npm install -g appium  # get appium
    > npm install wd         # get appium client
    > appium &               # start appium
    > node your-appium-test.js
